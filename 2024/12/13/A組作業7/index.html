A組回家功課

這篇文章主要是給我們班的同學使用，想要參考的同學可以來這裡看看!

第1題

題目

阿垂想要學習外語，發現某線上教學的網站正在推儲值點數促銷方案，如下:

原價: 1點/元

儲值點數 折扣
1~999 8 折
1000~2999 7 折
3000~4999 6 折
5000~ 5 折

請撰寫一程式，輸入儲值點數後，經計算其應付金額，
且使用if .. else if ..架構完成，輸出四個畫面結果。

解題思路

可以宣告一個變數，當用戶輸入的時候可以設定成0.8 0.7等等，然後再乘以就可以了。
理論上這題只要一個 < 就好，不用特別指定範圍，除非你是從大到小判斷。

程式碼

C++:

#include <iostream>
using namespace std;
int main()
{
    int points;
    float discount;
    cout << "請輸入欲儲值的點數: ";
    cin >> points;

    if (points < 1000) discount = 0.8;
    else if (points < 3000) discount = 0.7;
    else if (points < 5000) discount = 0.6;
    else discount = 0.5;

    cout << "儲值 " << points << " 點，應付 "<< (points * discount) << " 元"<< endl;
    return 0;
}

第2題

題目

聖誕節時某百貨公司促銷玩具禮盒，並將禮盒排列成錐形塔。每層的禮盒為
奇數個，已知頂層為1盒，寫下一層會比上一層多2盒。

(1)若銷售人員，只盤點最底層的禮盒數，要如何計算出該禮盒塔的禮盒總數
呢? 請以for迴圈設計之。
(2)若最底層的禮盒數的數量輸入錯誤，則顯示錯誤訊息，不予計算。

要求:
for 迴圈寫法，二個輸出畫面。 (A) 底層為 7 (B) 底層為 6,錯誤訊息
while 迴圈寫法，二個輸出畫面。 (A) 底層為 7 (B) 底層為 6,錯誤訊息

解題思路

這裡可以先檢查用戶輸入的值的餘數是不是等於0來判斷是不是奇數，然後因為頂層為1盒，所以宣告一個變數i為1，然後用個迴圈循環直到i == 用戶輸入值，且每次循環一次後i要加2。

程式碼

C++ for:

#include <iostream>
using namespace std;

int main()
{
    int number;
    int total = 0;
    cout << "請輸入最底層的禮盒數: ";
    cin >> number;

    if (!(number % 2)) {
        cout << "輸入錯誤，不予計算" << endl;
        return 0;
    }

    for (int i = 1; i <= number; i += 2) {
        total += i;
    }

    cout << total << endl;
    return 0;
}

C++ while:

#include <iostream>
using namespace std;

int main()
{
    int number;
    int i = 1;
    int total = 0;
    cout << "請輸入最底層的禮盒數: ";
    cin >> number;

    if (!(number % 2)) {
        cout << "輸入錯誤，不予計算" << endl;
        return 0;
    }

    while (i <= number) {
        total += i;
        i += 2;
    }

    cout << total << endl;
    return 0;
}

第3題

題目

元宵燈會時，主辦單位安排精彩的無人機表演，特別是分列式的花式表演，
組合出很多精彩的圖形，讓人印象深刻，原來組合圖形也可以由不同排列方式
來組成。
在這次燈會後，主辦單位希望藉由程式來進行不同形態的排列組合，由程式跑
出顯示圖形，作為下一期燈會籌畫的參考，以便讓無人機可以再做出更精彩多
變的表演。接下來請你先將下圖的分列式使用程式將圖形顯示出來，有關本次
程式設計所需遵守規範如下:

(1) 使用@符號來組成基本顯示圖形。
(2) 必須使用迴圈來完成。

要求:
for 迴圈寫法，三角形高度為 5
while 迴圈寫法，三角形高度為 12

解題思路

這裡要打印一個直角三角形，要用兩次的迴圈，第一層迴圈負責換行就好，第二層則要輸出@。
為甚麼這裡要用兩次迴圈?
因為第一行要打印@，第二行要打印@@依此類推。這樣不可以只使用一個迴圈，要有一個迴圈負責換行，另一個則負責打印@。

程式碼

C++ for:

#include <iostream>
using namespace std;

int main()
{
    int height;
    cout << "請輸入三角形的高: ";
    cin >> height;

    for (int i = 0; i < height; i++) {
        for (int j = 0; j <= i; j++) {
            cout << '@';
        }
        cout << endl;
    }

    return 0;
}

C++ while:

#include <iostream>
using namespace std;

int main()
{
    int height;
    int i = 0;
    cout << "請輸入三角形的高: ";
    cin >> height;

    while (i < height) {
        int j = 0; // 注意每次迴圈要設定j = 0
        while (j <= i) {
            cout << '@';
            j++;
        }

        cout << endl;
        i++;
    }

    return 0;
}
